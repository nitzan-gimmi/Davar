# רוג'ום הדוכיפת – GHC Core v1.0

"רוג'ום הדוכיפת אינו פרויקט. הוא התנגדות."

## מה קרה כאן?
משך שנים, השפה העברית סבלה מ"עיוורון מורפולוגי" – מודלי AI שמנפצים מילים עבריות לרסיסים חסרי משמעות (BPE). **רוג'ום הדוכיפת** (Hoopoe Collective) הוא מהפכה טכנולוגית ופילוסופית המשלבת:

- **HFBT** (Hebrew Factor-Based Tokenization): טוקניזציה המכבדת שורש ותבנית.
- **GHC** (Generative Hebrew Core): שכבת דריסה סימבולית לדקדוק מושלם.
- **וקטור הרצון (VoW)**: מנגנון הכרעה המאזן בין דיוק (HTQI), כוונה (IPI) ורגש (EAS).

## ארכיטקטורת המגאזורד (The Megazord)
המערכת פועלת על ארבעה צירים:

1. **ציר A (מורפולוגיה):** דיוק לשוני, HTQI >= 0.96.
2. **ציר B (פונולוגיה):** ניקוד והגייה (Nishpat).
3. **ציר C (כוונה):** זיהוי כוונת המשתמש (IPI).
4. **ציר D (קוגניציה):** ויסות רגשי (EAS).
